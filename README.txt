In FirstTask, the commutation plot for arbitrary molecules can be created and Pauli strings 
Are composed into families in an optimised way. 
Run the file 

CreateGraph.py 

In order for the graph and the respective cliques to be shown. 


In SecondTask, the block aggregation algorithm and its optimisation are carried out.
Run the file 

testScript.py 

To create the plots that are included in the report. There are furthermore animations of the 
Solving processes that are shown.
